# 功能指南（Features Guide）

版本：v2.0
最后更新：2026-03-16

## 1. 多模型智能路由

- 自动选择最合适的模型
- 低配设备自动降级
- 支持编程/长文本/多模态

## 2. 实时知识更新

- 提问时自动检索最新信息
- 支持 RSS 与网页抓取
- 可在设置中关闭

## 3. 知识图谱

- 记录人、事、物的关系
- 支持多跳推理
- 可在记忆管理中查看

## 4. 情绪神经网络

- 识别 6 种情绪
- 驱动语气与动画
- 可关闭情绪识别

## 5. UI 自动化

- 跨应用工作流
- 自动完成重复任务
- 支持自定义脚本
