# 功能指南

## 1. 多模型智能路由

作用：

- 根据任务类型、复杂度和硬件条件自动选择模型

用户可感知到的价值：

- 简单问题回复更快
- 复杂问题更稳定
- 低配置设备会自动降级，而不是直接失败

## 2. 实时知识更新

作用：

- 在需要最新信息时结合外部知识源进行补充

用户可控制：

- 是否允许联网
- 是否启用主动知识更新

## 3. 知识图谱与长期记忆

作用：

- 记录人、事、物和偏好之间的关系
- 帮助系统更好地理解长期上下文

常见使用场景：

- 记住偏好
- 关联人物与事件
- 在后续对话中进行关系推理

## 4. 情绪理解

作用：

- 分析文本、语音或其他线索中的情绪特征
- 调整回复语气、关怀方式和表现形式

## 5. 工具与自动化

作用：

- 帮助完成跨应用、多步骤任务
- 承担文件处理、信息整理、系统控制等工作
