# 常见问题

## Q1：没有 GPU 可以运行吗？

可以运行，但更适合轻量模型和基础场景，复杂推理和高质量多模态体验会明显受限。

## Q2：数据会自动上传到云端吗？

默认以本地优先为目标。只有在用户显式启用联网或云端增强能力时，才会发生对应的数据传输。

## Q3：如何删除记忆数据？

可通过记忆管理功能删除单条、按类型删除或清空全部历史数据。

## Q4：为什么有时回答更快、有时更慢？

Mirexs 会根据任务复杂度、资源状态和模型路由策略动态选择处理方式，因此不同任务的响应时间可能不同。

## Q5：首次启动为什么会比较慢？

首次启动通常涉及环境初始化、模型准备、配置加载和数据目录创建，因此会比后续启动更慢。
