# 安装指南（Installation Guide）

版本：v2.0
最后更新：2026-03-16

## 1. 安装方式

- Windows 安装包
- macOS 应用包
- Linux tar.gz 或包管理器

## 2. Windows 安装

1. 下载安装包
2. 双击安装并选择安装目录
3. 完成首次运行配置

## 3. macOS 安装

1. 打开 dmg 包
2. 拖拽到 Applications
3. 首次运行需允许权限

## 4. Linux 安装

```
chmod +x mirexs-installer.sh
./mirexs-installer.sh
```

## 5. 验证

- 打开应用，检查是否能加载模型
- 访问“系统状态”页面确认健康
