# 安装指南

## 1. 文档说明

本文件描述的是 Mirexs 目标安装体验。当前仓库更偏开发态，正式安装包与完整分发流程仍需后续交付。

## 2. 目标安装形态

- Windows 安装包
- macOS 应用包
- Linux 压缩包或系统包

## 3. 安装前检查

- 确认操作系统版本
- 确认磁盘空间是否足够
- 如需本地推理，确认 GPU 和驱动可用
- 如需知识图谱，确认是否准备安装 Neo4j

## 4. 安装后首次检查

- 应用是否能正常启动
- 模型目录是否可访问
- 数据目录和日志目录是否可写
- 麦克风、摄像头或输入监听权限是否按需授权

## 5. 当前仓库状态说明

当前仓库主要提供源码和开发文档，正式打包产物、安装器和发布流程需要后续补齐。
