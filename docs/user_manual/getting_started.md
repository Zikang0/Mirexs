# 快速上手（Getting Started）

版本：v2.0
最后更新：2026-03-16

## 1. 首次启动

1. 启动 Mirexs
2. 系统自动检测硬件并选择推荐模型
3. 根据提示完成初始化向导

初始化向导包含：
- 用户昵称与语言选择
- 是否启用长期记忆
- 是否允许联网知识更新

## 2. 基本交互

- 语音唤醒：说出唤醒词（默认“弥尔思”）
- 文本输入：在输入框直接输入
- 图像输入：拖拽或粘贴图片

## 3. 新手建议

- 先尝试简单任务，例如“总结今天日程”。
- 允许记忆功能可获得更个性化体验。

## 4. 示例对话

用户：帮我整理今天会议要点
Mirexs：已整理 5 个要点，并生成纪要草稿。
