# 快速上手

## 1. 适用对象

本指南面向首次使用 Mirexs 的用户，帮助在最短时间内完成初始设置并理解基本交互方式。

## 2. 首次启动流程

首次启动建议按以下顺序完成：

1. 启动应用或开发入口
2. 完成基础身份和语言设置
3. 检查系统硬件检测结果
4. 确认是否启用长期记忆
5. 确认是否启用联网知识更新

## 3. 初始化向导常见选项

- 用户昵称与语言
- 本地记忆是否启用
- 是否允许联网查询
- 默认交互方式：文本、语音或混合

## 4. 基本交互方式

- 文本输入：直接在输入框输入需求
- 语音输入：按住语音按钮或使用语音唤醒
- 图像输入：拖拽、上传或粘贴图片

## 5. 建议的新手任务

- “总结今天日程”
- “帮我整理会议纪要”
- “记住我喜欢周杰伦”
- “核查一下这条新闻”

## 6. 使用建议

- 初期先从单步任务开始，便于建立使用习惯
- 如需更强个性化体验，可开启长期记忆
- 如对隐私敏感，可先关闭联网能力，仅使用本地功能
