# 用户界面说明

## 1. 主界面结构

典型界面可分为以下区域：

- 会话区：显示输入、回复、证据和系统提示
- 控制区：模式、记忆、联网和模型相关开关
- 状态区：系统健康、资源占用和当前能力状态

## 2. 常见侧边功能

- 记忆管理
- 插件或工具管理
- 设备连接
- 设置与隐私控制

## 3. 交互原则

- 用户应能明确看见当前输入方式
- 涉及联网、记忆写入和高风险操作时，应给出清晰提示
- 降级、失败和等待状态不能只靠静默处理

## 4. 快捷操作示例

- 唤醒或聚焦输入框
- 搜索历史记录
- 打开帮助或命令面板

具体快捷键应以后续客户端实现为准。
