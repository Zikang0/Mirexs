# 故障排查

## 1. 启动失败

优先检查：

- Python 版本是否满足要求
- 依赖是否安装完整
- 配置文件是否缺失
- 数据目录和日志目录是否可写

## 2. 模型加载失败

优先检查：

- 显存和内存是否足够
- 模型路径是否存在
- 模型文件是否损坏
- 对应推理后端是否已安装

## 3. 语音相关问题

优先检查：

- 麦克风权限
- 音频设备是否可用
- 语音模型依赖是否安装

## 4. 联网或知识更新失败

优先检查：

- 网络连接
- 代理和防火墙
- 外部知识源是否可访问
- 是否在设置中关闭了联网能力

## 5. 界面卡顿或渲染过慢

优先检查：

- GPU 驱动
- 当前模型负载
- 是否启用了高质量 3D 渲染
- 是否可以切换到更轻量模式
