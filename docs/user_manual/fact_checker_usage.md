# 事实核查器使用指南

## 1. 功能说明

事实核查器用于在模型回答前后，对可验证信息进行交叉检查，减少明显事实错误和未经证实的陈述。

## 2. 典型使用方式

### 2.1 自动核查

在涉及新闻、日期、公开事实、政策或引用信息时，系统可以自动触发核查流程。

### 2.2 手动核查

用户可以直接提出：

- “核查一下这条新闻是否属实”
- “帮我验证这个说法”
- “给我证据来源”

## 3. 核查结果状态

| 状态 | 含义 |
|---|---|
| 已验证 | 多个可信来源一致支持 |
| 存疑 | 证据冲突或不足 |
| 不实 | 可信来源明确否定 |
| 待核查 | 需要进一步检索或处理 |

## 4. 如何理解结果

- 核查结论不等于绝对真理，而是基于当前证据的判断
- 对时效性很强的话题，应关注证据时间
- 对争议性议题，应重点看来源类型和冲突说明

## 5. 用户可要求查看的内容

- 主要证据来源
- 结论依据
- 冲突点说明
- 当前置信度或不确定性说明
