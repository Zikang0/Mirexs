# 事实核查器使用指南 (Fact Checker Usage)

## 1. 什么是事实核查器？ (What is Fact Checker?)

Mirexs 事实核查器是一个内置的智能系统，旨在确保 Mirexs 在与您交流时提供的信息是准确、可靠且经过验证的。它会自动从多个权威信源（如新闻、百科、官方报告）中检索并比对信息。

## 2. 如何使用？ (How to Use?)

### 2.1 自动核查 (Automatic Verification)

在大多数情况下，Mirexs 会在生成回复前自动运行事实核查器。如果发现信息存在争议或不确定，Mirexs 会在回复中向您提示。

### 2.2 手动请求核查 (Manual Request)

您可以直接要求 Mirexs 核查某条具体信息。例如：
*   “核查一下：2024年巴黎奥运会的开幕式是在塞纳河上举行的吗？”
*   “帮我验证这条新闻的真实性：[新闻链接或内容]”

## 3. 核查结果说明 (Understanding Results)

Mirexs 会为您提供以下几种核查状态：

| 状态 | 说明 |
| :--- | :--- |
| **已验证 (Verified)** | 该信息与多个权威信源一致，可信度极高。 |
| **存疑 (Uncertain)** | 不同信源之间存在冲突，或缺乏足够的证据进行验证。 |
| **不实 (False)** | 该信息已被权威信源明确否定。 |
| **待核查 (Pending)** | 正在进行深度核查，稍后将提供结果。 |

## 4. 查看证据 (Viewing Evidence)

对于重要的核查结果，您可以要求 Mirexs 展示证据来源。Mirexs 会列出相关的信源名称、链接以及简要的证据描述。

## 5. 常见问题 (FAQ)

*   **Q: 事实核查器是 100% 准确的吗？**
    *   A: 虽然我们尽力确保准确性，但事实核查器依赖于外部信源。对于极具争议或实时发生的新闻，建议您结合多个渠道进行判断。
*   **Q: 我可以添加自己的信源吗？**
    *   A: 目前该功能仅面向开发者开放。如果您有推荐的权威信源，欢迎通过反馈渠道告诉我们。
