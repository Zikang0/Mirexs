# 形象自定义

## 1. 目标

形象自定义用于让用户调整 Mirexs 的视觉外观、声音风格和部分行为表现，使交互更具专属感。

## 2. 可调整项

- 主题颜色
- 角色外观
- 配饰与服装
- 语音音色
- 动作或表情风格

## 3. 预设管理

建议支持：

- 保存多个预设
- 一键切换预设
- 导入导出预设

## 4. 使用注意事项

- 高精度模型和复杂特效可能增加渲染负载
- 语音音色切换可能需要额外模型或资源
