# 知识与记忆管理

## 1. 记忆类型

- 情景记忆：具体事件、对话和时间相关信息
- 语义记忆：稳定偏好、常识和概念总结
- 程序记忆：习惯流程、常用操作和工作模式

## 2. 常见操作

- 查看已保存的记忆
- 搜索和筛选
- 编辑或删除
- 导出或清空

## 3. 使用建议

- 对隐私敏感的用户可以关闭长期记忆
- 对需要个性化体验的用户建议保留偏好和重要长期信息
- 删除记忆前应确认是否会影响个性化能力

## 4. 隐私控制

- 可按类型开关记忆
- 可限制联网或跨设备同步
- 支持清理全部历史数据
