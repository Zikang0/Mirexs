# 贡献指南（Contribution Guide）

版本：v2.0
最后更新：2026-03-16

## 1. 贡献方式

- 提交 Issue
- 提交 PR
- 完善文档
- 提供插件

## 2. Issue 分类

- Bug
- Feature Request
- 文档改进

## 3. 分支规范

- `main`：稳定分支
- `dev`：开发分支
- 功能分支：`feature/<name>`
- 修复分支：`fix/<name>`

## 4. 提交流程

1. Fork 仓库
2. 创建功能分支
3. 提交变更并通过测试
4. 提交 PR 并等待评审

## 5. Commit 规范

- 建议使用 Conventional Commits
- 示例：`feat: add model router`
