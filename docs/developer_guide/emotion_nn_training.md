# 情绪模型训练指南

## 1. 训练目标

Mirexs 的情绪模型目标不是简单做文本分类，而是为后续的回复语气、主动行为和多模态反馈提供稳定的情绪判断基础。

当前建议目标包括：

- 识别 6 类基础情绪
- 支持中性和模糊场景降级
- 支持后续个性化微调

## 2. 数据要求

可用数据来源包括：

- 文本情绪标注语料
- 语音情绪数据
- 面部表情数据
- 用户授权后的个性化反馈数据

强制要求：

- 数据必须匿名化
- 必须有合法授权或公开许可
- 训练集、验证集、测试集必须分离

## 3. 特征建议

- 文本：句向量或 Transformer 编码结果
- 语音：声学特征、韵律特征
- 表情：关键点、表情 embedding
- 多模态融合时要统一时间窗口和标签口径

## 4. 训练流程

1. 数据清洗与标签校验
2. 特征提取与归一化
3. 训练基础分类器
4. 验证误分类样本
5. 校准阈值和置信度输出
6. 导出模型与推理配置

## 5. 输出要求

训练结果至少要包含：

- 模型权重文件
- 标签映射
- 推理阈值配置
- 训练数据说明
- 评估结果摘要

## 6. 评估指标

- Accuracy
- Macro F1
- 每类召回率
- 低置信样本比例

## 7. 与系统集成时要注意

- 情绪输出必须支持“不确定”或低置信回退
- 不允许高风险场景只依赖单一情绪预测做关键决策
- 推理输出应能被回复生成、行为引擎和 UI 使用
