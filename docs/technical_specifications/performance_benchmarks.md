# 性能基准规范（Performance Benchmarks）

版本：v2.0
最后更新：2026-03-16

## 1. 基准测试目标

- 确保对话响应满足 1s 内体验
- 确保复杂推理在 3‑5s 内完成
- 确保知识检索与图谱推理为实时级

## 2. 测试环境

- CPU：i7/Ryzen7
- GPU：RTX 3060 / RTX 4090
- 内存：32GB / 64GB
- 存储：NVMe SSD

## 3. 指标定义

- P50/P95 延迟
- QPS
- 错误率
- GPU 显存峰值

## 4. 场景

- 场景 A：日常聊天（短文本）
- 场景 B：复杂推理（长上下文）
- 场景 C：多模态任务
- 场景 D：图谱推理

## 5. 判定标准

- 80% 请求满足 SLA
- 失败率 < 1%

---

本规范为契约优先文档。
