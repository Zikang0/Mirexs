# 系统需求规范（System Requirements）

版本：v2.0
最后更新：2026-03-16

## 1. 目标

定义 Mirexs 的运行环境基线与推荐配置，用于交付、部署评估与硬件采购。

## 2. 运行环境分级

| 级别 | 目标场景 | CPU | 内存 | GPU | 存储 |
|---|---|---|---|---|---|
| 轻量版 | 基础聊天 | 4 核 | 16GB | 可选 | 100GB SSD |
| 推荐版 | 日常多任务 | 8 核 | 32GB | RTX 3060 12GB | 500GB SSD |
| 高性能 | 70B/多模型 | 16 核 | 64GB | RTX 4090 24GB | 1TB NVMe |

## 3. 操作系统支持

- Windows 10/11
- Ubuntu 20.04/22.04
- macOS 12/14（受 Metal 约束）

## 4. 关键系统依赖

- CUDA 12.x（NVIDIA GPU）
- cuDNN 8.x
- Java 17+（Neo4j）
- FFmpeg（音视频处理）
- Git LFS（模型下载）

## 5. 端口与服务依赖

| 服务 | 默认端口 |
|---|---|
| REST API | 8000 |
| WebSocket | 8001 |
| Neo4j | 7474/7687 |
| Redis | 6379 |
| InfluxDB | 8086 |

## 6. 存储规划

- 模型文件：8B 模型 5‑8GB；32B 模型 20‑30GB；70B 模型 40‑90GB。
- 数据库：建议预留 50‑200GB。

## 7. 权限与安全

- Windows 需允许麦克风、摄像头权限
- macOS 需在系统设置中允许输入监听

---

本规范为契约优先文档，作为交付与部署验收的统一依据。
