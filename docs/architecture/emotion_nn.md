# 情绪神经网络（Emotion NN）架构规范

版本：v2.0
最后更新：2026-03-16

## 1. 目标

情绪神经网络用于识别用户情绪，驱动回应语气、动画与主动关怀。

## 2. 情绪分类

- 开心（happy）
- 悲伤（sad）
- 愤怒（anger）
- 恐惧（fear）
- 惊讶（surprise）
- 平静（calm）

## 3. 模型结构

- 输入：文本向量（Sentence‑Transformers 384 维）
- 网络：BiLSTM + Attention
- 输出：6 类概率分布

## 4. 多模态扩展

- 语音特征：能量、音高、语速
- 视觉特征：AU 单元、表情关键点
- 融合策略：Late Fusion

## 5. 推理流程

1. 文本/语音/视觉预处理
2. 特征融合
3. 模型推理
4. 置信度校准
5. 低置信度回退（规则引擎）

## 6. 训练流程

- 数据收集 → 清洗 → 标注
- 基础模型训练
- 用户个性化微调

## 7. 输出格式

```json
{
  "emotion": "sad",
  "probabilities": {
    "happy": 0.05,
    "sad": 0.62,
    "anger": 0.07,
    "fear": 0.08,
    "surprise": 0.03,
    "calm": 0.15
  },
  "confidence": 0.62
}
```

## 8. 性能目标

- 推理延迟 < 50ms
- 准确率 > 85%

---

本规范为契约优先文档。
