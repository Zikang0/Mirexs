# Mirexs 定价模型 (Pricing Model)

## 1. 定价策略概述 (Pricing Strategy)

Mirexs 采用 **Freemium (基础免费 + 高级订阅)** 的定价模型，旨在降低用户入门门槛，同时通过高级功能和服务实现商业化。

## 2. 订阅方案 (Subscription Plans)

| 功能 | 基础版 (Free) | 专业版 (Pro) | 旗舰版 (Ultimate) |
| :--- | :--- | :--- | :--- |
| **核心模型** | Llama 3.1 8B / Qwen 2.5 7B | Llama 3.1 70B / Qwen 3.5 32B | DeepSeek V3 / Llama 4 Maverick |
| **记忆容量** | 1,000 条对话记录 | 无限对话记录 | 无限对话记录 + 深度知识图谱 |
| **3D 形象** | 基础猫咪形象 | 全部猫咪形象 + 基础自定义 | 全面自定义 + AI 形象生成 |
| **主动行为** | 基础提醒 | 全面主动关怀 | 深度个性化主动行为 |
| **推理节点** | 公共节点 (排队) | 优先节点 | 专属私有节点 |
| **价格** | $0 / 月 | $19.9 / 月 | $49.9 / 月 |

## 3. 增值服务 (Add-on Services)

*   **形象资产包**：单次购买特定的皮肤、动作或场景装饰（$0.99 - $9.99）。
*   **模型微调服务**：为用户定制专属的个性化模型适配器（$99 / 次）。
*   **API 调用额度**：针对开发者的按量计费接口（$0.01 / 1k tokens）。

## 4. 企业级方案 (Enterprise Solutions)

*   **定制化数字员工**：根据企业品牌形象和业务流程定制。
*   **私有化部署**：支持在企业内部服务器或私有云部署。
*   **高级安全审计**：提供符合行业标准的合规性报告。
*   **价格**：需联系销售团队进行定制化报价。

## 5. 优惠与促销 (Discounts & Promotions)

*   **早鸟优惠**：项目上线首月订阅专业版可享受 5 折优惠。
*   **学生优惠**：凭有效学生证件可享受 7 折优惠。
*   **年度订阅**：一次性订阅一年可享受 8 折优惠。
