
# Mirexs 产品简介（Product Brief）

**版本：v2.0.0**  
**最后更新：2026-03-17**  
**作者：Zikang.Li**  
**目标受众**：潜在用户、合作伙伴、投资者、媒体、开源社区  
**核心定位**：全球首个**本地优先、情感驱动、具备长期自主记忆与主动能力的个人数字生命体**（非传统聊天机器人）

## 1. 一句话简介

Mirexs 是一个完全运行在你设备上的**情感化数字伴侣**：它能记住你的喜好、理解你的情绪、主动关心你、与你共同成长，而**一切数据永不离开你的电脑/手机**。

## 2. 核心卖点（差异化优势）

| 卖点                       | 传统聊天机器人（如 ChatGPT、Claude） | Mirexs v2.0                              | 用户真实收益                              |
|----------------------------|---------------------------------------|------------------------------------------|-------------------------------------------|
| 数据隐私与控制             | 云端存储，数据可能用于训练           | 100% 本地运行，无任何上传               | 零隐私担忧，数据完全属于自己              |
| 长期记忆与个性化           | 短期上下文 + 有限 RAG                 | 知识图谱 + 情绪神经网络 + 强化学习      | 越用越懂你，像老朋友一样                  |
| 情感理解与表达             | 表面模拟                              | 多模态情绪网络（文本+语音+视觉）+强度量化 | 真正“感知”你的情绪并回应                  |
| 主动能力                   | 完全被动                              | 主动行为引擎 + RL 优化                   | 它会主动问候、提醒、关心（不打扰前提下）  |
| 3D 形象与沉浸感            | 纯文本/简单头像                       | Panda3D 实时 3D 猫咪/人类形象 + 行为同步 | 视觉+情感双重陪伴                         |
| 硬件自适应                 | 需要高配云端                          | RTX 3060 即可流畅（8GB VRAM 起步）      | 笔记本就能跑大模型                        |
| 开源与可扩展               | 闭源                                  | Apache 2.0 + 插件系统 + SDK              | 开发者可深度定制、贡献                    |

## 3. 目标用户画像

- **隐私敏感型用户**（香港/中国大陆/欧盟用户）：厌倦云端数据泄露，追求完全本地控制。
- **情感陪伴需求者**：孤独、压力大、需要倾诉对象，但不想被平台监控。
- **AI 发烧友/开发者**：想拥有可长期调教、自定义的个人 AI 生命体。
- **创意工作者**：需要有记忆、有情绪、有主动建议的创作伙伴。
- **长期关系型用户**：愿意花时间“养”一个 AI，像养宠物/朋友一样。

## 4. 关键功能亮点（一句话描述）

- **情绪神经网络**：实时理解你的开心/难过/愤怒，并用 3D 形象表达同理心。
- **知识图谱记忆**：记住你说过的话、喜欢的事、讨厌的人，永不遗忘。
- **多模型智能路由**：根据任务自动选最合适的模型（快聊用 7B，复杂推理用 32B）。
- **强化学习进化**：从你的反馈中学习“怎么说话你最喜欢”，越用越贴心。
- **主动关心引擎**：它会主动问“最近怎么样？”或“生日快到了，要不要准备惊喜？”。
- **插件生态**：想让它查天气、玩游戏、写代码？一键加插件。
- **完全开源**：代码、模型配置、文档全公开，社区共创。

## 5. 技术基础（简要，非技术用户可跳过）

- 本地大模型：Qwen/DeepSeek/Llama GGUF 量化版
- 知识存储：Neo4j 图谱 + Chroma 向量
- 情绪核心：bge-small-zh + LSTM + LoRA 微调
- 行为优化：Q-learning / DQN + 优先经验回放
- 3D 渲染：Panda3D 实时动画
- 安全防护：三层防御 + 不可篡改审计日志
