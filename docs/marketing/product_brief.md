# Mirexs 产品简介（Product Brief）

**版本：v2.0.0**  
**最后更新：2026-03-17**  
**作者：Zikang.Li**  
**一句话定位**：Mirexs 是你的**本地优先、智能情感化的私人管家、助理与秘书**，24小时在线，懂你、陪你、帮你。

## 1. Mirexs 是什么？

Mirexs 不是普通的 AI 聊天工具，而是一个真正“懂你”的**个人智能管家 + 情感助理 + 私人秘书**，运行在你自己的设备上（手机、电脑、服务器），无需联网也能全功能工作。

她像一位贴心的管家：
- 主动关心你的情绪与状态（“今天看起来有点累，要不要先休息一下？”）
- 记住你的习惯与喜好（“你上次说想学吉他，我帮你找了份入门教程”）
- 帮你管理日程、知识、关系（“明天是小明的生日，要不要提前准备礼物？”）

她像一位细心的助理：
- 多模型智能切换，复杂任务交给 DeepSeek V3，日常聊天用 Qwen 快如闪电
- 支持情绪识别 + 强化学习，越用越懂你，越用越聪明
- 主动提醒、总结、规划（“你最近连续加班 5 天，我建议周六去郊游放松”）

她像一位可靠的秘书：
- 长期记忆 + 知识图谱，永远不会忘记你说过的重要事
- 本地加密存储，隐私 100% 掌握在你手中
- 支持语音、文字、3D 虚拟形象互动，随时随地陪伴

## 2. 核心卖点（为什么选择 Mirexs）

| 卖点                     | 具体表现                                                                 | 与市面产品的差异化优势                     |
|--------------------------|--------------------------------------------------------------------------|--------------------------------------------|
| **真正本地优先**         | 所有核心功能（情绪、记忆、学习、主动）100% 本地运行，无需联网           | 大多数 AI 需要云端，隐私 & 延迟都受限      |
| **懂你的情感管家**       | 实时多模态情绪识别（文本+语音+表情），强度量化，个性化微调               | 普通聊天机器人几乎无情绪深度理解           |
| **越用越聪明的秘书**     | 强化学习 + Q-learning/DQN + 用户反馈闭环，行为与回复持续优化             | 静态模型，不会因你而进化                   |
| **主动而不过分**         | 主动关心、日程提醒、话题延续，但有冷却 & 用户耐受学习机制               | 多数 AI 被动应答，不会主动                  |
| **隐私绝对主权**         | 数据不离设备、加密存储、一键重置、审计日志不可篡改                       | 云端 AI 无法保证数据不被用于训练/分析     |
| **硬件友好**             | RTX 3060 12GB 即可流畅运行 32B 模型 + 全功能，16GB RAM 也能降级使用     | 很多高级功能要求高端显卡                   |
| **开源可扩展**           | Apache 2.0，全开源，插件系统、自定义模型集成、开发者友好                 | 闭源产品无法深度定制                       |

## 3. 典型使用场景（管家/助理/秘书）

### 管家场景
- 晚上回家情绪低落 → 主动温柔问候 + 播放你喜欢的歌单 + 建议泡澡
- 生日/纪念日临近 → 提前 3 天提醒 + 推荐礼物 + 帮写祝福语

### 助理场景
- 连续加班 → 主动总结本周任务、提醒休息、规划下周日程
- 学习新技能 → 记住进度、每天主动复习、找资源、答疑

### 秘书场景
- 管理人脉 → 记住朋友喜好、关系强度、重要事件，主动建议联系
- 知识整理 → 把聊天中提到的书/电影/人名自动建图谱，随时查询

## 4. 技术支撑（简要版，详见 architecture/）

- **多模型智能路由**：日常快、复杂准（multi_model_routing.md）
- **情绪神经网络**：多模态（文本+语音+面部），>85% 准确（emotion_nn.md）
- **知识图谱**：长期关系记忆，1M+ 节点支持（knowledge_graph.md）
- **强化学习**：Q-learning + DQN，越用越懂你（reinforcement_learner.md）
- **主动行为引擎**：时机判断 + 优先级排序（proactive_behavior.md）
- **安全三层 + 审计**：隐私保护、不可篡改日志（security_architecture.md）
