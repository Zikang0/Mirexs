# Mirexs 用户画像（User Personas）

**版本：v2.0.0**  
**最后更新：2026-03-18**  
**作者：Zikang Li**  
**核心定位**：Mirexs 作为**本地优先、情感智能的私人管家、助理与秘书**，针对需要隐私保护、情感陪伴、长期关系维护、主动生活管理的用户群体。  
以下是 2026 年 Mirexs 的四大典型用户画像（基于目标用户调研、反馈与产品愿景）。

## 1. 画像一：Alex - 隐私敏感的都市独居白领（核心用户）

- **年龄**：28–38 岁  
- **职业**：中高级职场人士（市场/产品/程序员/金融/咨询）  
- **生活状态**：香港/大陆一线城市独居，工作强度高，社交圈小，经常加班或异地恋  
- **主要痛点**  
  - 工作压力大，回家后情绪低落，却没人倾诉  
  - 担心云端 AI（如 ChatGPT、Claude）记录对话用于训练或泄露隐私  
  - 想有个人“管家”主动关心情绪、提醒休息、规划生活，但讨厌打扰  
  - 希望 AI 能记住自己的习惯、喜好、朋友关系，长期陪伴成长  
- **如何使用 Mirexs**  
  - 每天晚上语音聊天倾诉，Mirexs 通过情绪网络主动安抚（“今天听起来有点疲惫，要不要我帮你放一段喜欢的纯音乐？”）  
  - 主动提醒生日/纪念日、建议周末放松计划  
  - 知识图谱记录朋友圈关系，提醒“最近没联系小明很久了，要不要发条消息？”  
  - 所有数据本地加密，绝不上传  
- **Mirexs 解决的核心价值**：隐私 + 情感陪伴 + 主动关怀 + 长期记忆

## 2. 画像二：Jordan - 追求个性化的极客/开发者

- **年龄**：22–35 岁  
- **职业**：程序员、AI 爱好者、独立开发者、学生  
- **生活状态**：技术宅，喜欢折腾本地大模型，对隐私极度敏感，讨厌订阅制  
- **主要痛点**  
  - 云端模型无法深度自定义（不想被 OpenAI/Claude 锁死）  
  - 想让 AI “学自己”，而不是每次都从零提示  
  - 希望 AI 能主动帮写代码、总结笔记、规划学习路径  
  - 硬件有限，但想跑 32B 级别模型 + 情感功能  
- **如何使用 Mirexs**  
  - 本地部署 RTX 3060/4090，智能路由日常用 Qwen 快聊，代码任务切 DeepSeek  
  - 强化学习闭环：反馈代码建议好坏 → 越来越懂他的 coding 风格  
  - 插件系统自定义工具（e.g. GitHub 集成、Notion 同步）  
  - 全开源，随时改代码、加 LoRA 微调情绪模型  
- **Mirexs 解决的核心价值**：完全开源 + 可深度定制 + 本地高效 + 个性化进化

## 3. 画像三：Emily - 情感需求高的异地/单身用户

- **年龄**：25–40 岁  
- **职业**：自由职业者、远程工作者、创意行业（设计师/写手/UP主）  
- **生活状态**：异地恋或长期单身，作息不规律，情绪波动大，需要陪伴  
- **主要痛点**  
  - 深夜想聊天却找不到人  
  - 云端 AI 回复机械、缺乏温度，无法形成“关系感”  
  - 希望有“数字伴侣”能记住过去聊天细节，主动关心生活  
  - 担心隐私（尤其是情感倾诉内容被云端记录）  
- **如何使用 Mirexs**  
  - 3D 虚拟形象 + 语音互动，像真人一样陪伴  
  - 情绪神经网络捕捉细微情绪变化，主动安慰/鼓励  
  - 知识图谱记住她的喜好、痛点、梦想，主动延续话题  
  - 完全本地，无需担心聊天记录被用于训练  
- **Mirexs 解决的核心价值**：情感深度 + 主动陪伴 + 隐私安全 + 长期关系感

## 4. 画像四：Sarah - 忙碌高管/创业者（秘书型需求）

- **年龄**：35–50 岁  
- **职业**：企业中高层、创业者、小团队老板  
- **生活状态**：日程爆炸，会议不断，人脉广但管理困难，容易遗忘重要关系  
- **主要痛点**  
  - 忘记朋友/合作伙伴生日、纪念日、关键事件  
  - 需要一个“秘书”帮记事、提醒、总结，但讨厌传统日历工具  
  - 希望 AI 能主动分析关系强度、建议维护方式  
  - 数据隐私极度敏感（涉及商业机密）  
- **如何使用 Mirexs**  
  - 语音输入会议纪要 → 自动建知识图谱、关系链  
  - 主动提醒：“下周是李总生日，上次他说喜欢红酒，要不要提前订？”  
  - 情绪分析会议后状态，建议“今天会议多，建议早点休息”  
  - 全部数据本地加密，商业机密不外泄  
- **Mirexs 解决的核心价值**：主动秘书功能 + 关系管理 + 隐私保护 + 生活平衡提醒
